# 📚 Coding Test Study Repository

코딩 테스트 대비를 위해 알고리즘 문제 풀이와 교재 학습 내용을 기록하는 저장소입니다.
문제 풀이 기록과 학습 과정을 정리하여 **지속적인 알고리즘 학습 아카이브**로 관리합니다.

# ✏️ Commit Convention

학습 기록을 명확하게 남기기 위해 다음과 같은 커밋 규칙을 사용합니다.

## Commit Format

```
type: [Book] Chapter - 내용
```

### Example

```
study: [Doit] Ch02 배열과 리스트 예제 구현
study: [Doit] Ch03 구간 합 알고리즘 구현
study: [이코테] Ch05 DFS 기본 구현
practice: [Doit] Ch03 연습 문제 풀이
refactor: [Doit] Ch05 BFS 코드 개선
```

---

## Commit Type

| Type         | Description      |
| ------------ | ---------------- |
| **study**    | 책 내용을 따라 알고리즘 구현 |
| **practice** | 장 연습 문제 풀이       |
| **example**  | 책 예제 코드 작성       |
| **refactor** | 코드 개선 또는 재작성     |
| **docs**     | README 및 문서 수정   |

---

## Commit Example

```
study: [Doit] Ch03 Prefix Sum 구현
practice: [Doit] Ch03 구간 합 연습 문제
study: [이코테] DFS 구현
```
